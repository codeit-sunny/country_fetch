# Country Info Explorer 🌍
## Project Overview
Country Info Explorer is a simple web project that fetches and displays information about countries using an API.
The application shows important details such as the country name, flag, currency, population, and area.
## Features
* Fetch country data from an API
* Display country flag
* Show common country name
* Display currency information
* Show population of the country
* Display total area of the country
* Simple and clean user interface
## Technologies Used
* HTML
* CSS
* JavaScript
* REST API
## How It Works
The application fetches country data from a public API and dynamically displays the information on the webpage.
## How to Run the Project
1. Download or clone this repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.
## API Used
This project uses the **REST Countries API** to fetch country information.
